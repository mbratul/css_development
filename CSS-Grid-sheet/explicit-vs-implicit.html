CSS Grid

Explicit and implicit tracks

Notice when you define only grid-template-column without defining rows, there are rows will be created automatically by the browser which is part of the implicit grid. Whereas the explicit grid consists of any rows and columns defined with grid-template-columns or grid-template-rows
So explicit track is defined by you.
and implicit track is defined by the browse.
