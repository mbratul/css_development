CSS Grid

auto-fill & auto-fit

By trying auto-fill and auto-fit values for the first time, they might looks the same that we have flexible number of columns that achieve a responsive grid. but by inspecting the grid container and looking into the grid lines numbers, you see that:
auto-fill that sorting the grid items next to each others and keep the columns lines to be automatically created based on the size left. The last column line (-1) will be the the end of the container.
auto-fit Also sorting the grid items next to each others. However, looking into the grid column lines, will see that last line (-1) will end up on the last item we have in the container.
