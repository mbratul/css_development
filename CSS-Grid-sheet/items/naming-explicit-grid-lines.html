CSS Grid

Naming explicit grid lines

We can define names explicitly for our grid lines.
For example when we define our columns by
grid-template-columns: 1fr 1fr 1fr fr 1fr; , we can name the first and last line for example this way
grid-template-columns: [outer-start] 1fr 1fr 1fr fr 1fr [outer-end]; .
