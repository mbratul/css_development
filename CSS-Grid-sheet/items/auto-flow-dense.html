CSS Grid

auto-flow (dense)

Once defining the grid for the container, By default the items flow in rows. Using grid-auto-flow can controll the flow of items.
It takes values like:
grid-auto-flow: row;
grid-auto-flow: column;
grid-auto-flow: dense;
grid-auto-flow: row dense;
grid-auto-flow: column dense;
