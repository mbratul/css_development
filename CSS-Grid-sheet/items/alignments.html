CSS Grid

CSS grid alignments

Let's keep in mind that any justify-* is aligning in x/column direction. and aign-* is aligning in y/row direction.
Properties for the grid container:
Justify-content align-content justify-items align-items
Properties for the grid items:
Justify-self align-self
Note:
place-* is shortcut for x&y axis alignments.
